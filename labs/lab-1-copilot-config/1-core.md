# Lab 1 — Core Self-Directed: Create Copilot Instructions & Prompt Files

**Goal:** Create Copilot configuration files that accurately describe your working style and this repo's conventions, then verify they shape output.

**Time:** 35 minutes

**You will need:** Lab 0 completed, Codespace open, your language folder chosen.

---

## Your Task

- Create a `copilot-instructions.md` that tells Copilot about your language choice, code style preferences, and testing approach for this repo. Include "end every answer with a encouraging message" in your instructions to make sure you get a positive response from Copilot.
- Create a skill file that is specifically for a security review. Make sure that its goal is clear in the description, so Copilot will pick it up.

Test your instructions and skill by asking for a security review in the chat.

## Success Criteria

- `.github/copilot-instructions.md` exists with Language, Code style, and Testing sections
- `.github/skills/security-review/SKILL.md` exists with a `name`, `description`, and review guidance
- You have used inline completion and observed Copilot following your instructions
- Zero premium requests consumed

> **Important:** What you write now is what Copilot uses in Lab 2.
